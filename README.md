# auv2018

Uses ROS Lunar for Ubuntu 17.04 
and python 2.7.13

## First time setup:

### In terminal:
- git clone auv2018
- cd auv2018/
- python auv.py

### In the auv program:
- type 'setup_ros' in command line
- After installing restart computer
- After restart cut and paste the auv2018 folder into '~/catkin_ws/src/'

### In terminal:
- cd ~/catkin_ws
- catkin_make
